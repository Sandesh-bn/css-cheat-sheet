Attribute selectors used to target elements that has an attribute.
img[alt] targets all element that has alt attribute
a[title='corporate'] targets all a that has title = 'corporate'
a[title~='blue'] blue can be anywhere in the title. not just the beginnig
a[title^='blue'] target a if title starts it 'blue'
a[title$='blue'] target a if title ends with 'blue'
